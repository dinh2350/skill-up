## ğŸ› ï¸ Topics Breakdown

### HTML (2-3 weeks)

| Topic | Key Concepts |
|-------|--------------|
| Document Structure | DOCTYPE, head, body, meta tags |
| Text Elements | Headings, paragraphs, formatting |
| Links & Navigation | Anchors, navigation structure |
| Media | Images, video, audio, iframes |
| Lists & Tables | Ordered, unordered, data tables |
| Forms | Input types, validation, accessibility |
| Semantic HTML5 | header, nav, main, article, section, aside, footer |
| SEO Basics | Meta tags, Open Graph, structured data |
| Accessibility | ARIA, keyboard navigation, screen readers |

### CSS (3-4 weeks)

| Topic | Key Concepts |
|-------|--------------|
| Selectors | Element, class, ID, attribute, combinators |
| Specificity | Cascade, inheritance, !important |
| Box Model | Margin, padding, border, box-sizing |
| Colors & Backgrounds | Hex, RGB, HSL, gradients |
| Typography | Fonts, text properties, web fonts |
| Display & Position | Block, inline, relative, absolute, fixed, sticky |
| Flexbox | Container, items, alignment, ordering |
| CSS Grid | Template areas, tracks, responsive grids |
| Responsive Design | Media queries, mobile-first, fluid layouts |
| Animations | Transitions, keyframes, transforms |
| CSS Variables | Custom properties, theming |
| Pseudo-classes/elements | :hover, :focus, ::before, ::after |

### JavaScript (4-6 weeks)

| Topic | Key Concepts |
|-------|--------------|
| Variables | let, const, var, scope |
| Data Types | Primitives, objects, type coercion |
| Operators | Arithmetic, comparison, logical |
| Control Flow | if/else, switch, loops |
| Functions | Declaration, expression, arrow, closures |
| Arrays | Methods, iteration, destructuring |
| Objects | Properties, methods, this keyword |
| DOM | Selection, manipulation, traversal |
| Events | Listeners, event object, delegation |
| ES6+ | Template literals, spread/rest, classes, modules |
| Async | Callbacks, promises, async/await, fetch |
| Error Handling | try/catch, throwing errors |
| Web APIs | localStorage, URL, Date, timers |
